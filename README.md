# 🍸 Cocktail Finder

A web application built with Express.js and Axios that fetches random cocktail recipes from TheCocktailDB API and displays them in a modern UI with dynamic background effects.

---

## 📌 Features

- Express server setup
- Public API integration (TheCocktailDB)
- Axios HTTP requests
- Dynamic rendering using EJS
- Blurred background based on cocktail image
- Error handling
- Responsive, styled UI

---

## 🛠 Technologies Used

- Node.js
- Express.js
- Axios
- EJS
- CSS
